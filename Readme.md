# Instructions to use the application
* Use browser to open http://localhost:8080/


You can use the application normally; you don't need to type anything extra if you want to 
go to another page.

Create new fruit:
* On the homepage click on "Obst list" 
* Click on "Add new fruit"
* Fill in the form and click on "Save"

Edit existing fruit:
* On the homepage click on "Obst list"
* Click on the edit button of the fruit you want to edit
* Edit the form and click on "Save"

Delete existing fruit:
* On the homepage click on "Obst list"
* Click on the delete button of the fruit you want to delete
